# Frontend Template

Reusable frontend starter structure for another software project.

## Suggested Structure

- `src/` UI code, pages, hooks, utilities
- `components/` shared reusable components
- `public/` static assets
- `styles/` theme tokens and global styles

## Notes

- Copy this folder as the base for a new frontend.
- Keep API clients and auth helpers centralized.
- Reuse the same design language across modules.
